## Instructions

npx hardhat compile

Create dev accounts:

npx hardhat node

Keep node running, in new window:

npx hardhat run scripts/deploy.js --network localhost

Install Metamask Chrome (Brave) extension

npx hardhat run scripts/deploy.js --network ropsten
